Aula 18/03/2025 - Variáveis em JavaScript

  1. Declare uma variável com seu nome:
    // Exemplo:
    var nome = "Francisco";
    console.log(nome);
  2. Declare uma variável com a sua idade:
    var idade = 59;
    console.log(idade);
  3. Declare uma variável com a sua comida preferida:
    var comida = "Pizza";
    console.log(comida);
  4. Declare 5 variáveis com diferentes valores:
    var x = 10, y = 20, z = 5, a = "valor", b = "total";
    console.log(x, y, z, a, b);
